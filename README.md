# Jogo de Adivinhação 

Projeto simples em Python para demonstração ao professor.
O usuário tenta adivinhar um número secreto escolhido pelo computador.

## Como funciona

* O computador escolhe um número aleatório entre **1 e 100**.
* O jogador tenta adivinhar.
* O programa informa se o palpite é **maior**, **menor** ou **correto**.
* Conta quantas tentativas foram feitas.

## Como rodar

1. Instale o Python 3.
2. Execute no terminal:

```
python main.py
```

## Exemplo de saída

```
Tente adivinhar o número entre 1 e 100!
Seu palpite: 50
Muito alto!
Seu palpite: 20
Muito baixo!
Seu palpite: 37
Parabéns! Você acertou em 3 tentativas.
```
